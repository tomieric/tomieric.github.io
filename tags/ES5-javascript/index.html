nodeList-forEach

10月 11, 2014 发布在 JavaScript

nodeList-forEach

描述

forEach 方法按升序为数组中的每个有值的元素执行一次给定的 callback
函数，只有在那些有值的索引上才会调用 callback 函数，那些被删除掉的索引或者从未赋值过的索引将会被跳过。

callback 函数会被依次传入三个参数：

  • 元素值
  • 元素索引
  • 被遍历的数组对象本身
